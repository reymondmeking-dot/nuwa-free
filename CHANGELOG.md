# Changelog

## 1.2.0 — 2026-07-12

- 页面改为从 `nuwa-free-tools.json` 单一数据源加载，删除重复硬编码工具列表。
- 新增名称/用途搜索、分类筛选、可分享筛选参数和安全 DOM 渲染。
- 拆分语义化 HTML、响应式 CSS 与模块脚本，改善可访问性和移动端体验。
- `build` 现在生成仅含公开文件的 `dist/`，GitHub Pages 只发布该目录。
- 预览服务默认绑定 `127.0.0.1`，只服务构建后的白名单普通文件，并拒绝符号链接和越界路径。
- 新增真实 HTML 检查与统一 `npm run check` 流程。
